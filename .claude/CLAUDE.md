# Claude Code Instructions

This file contains project-specific instructions for Claude Code to follow when working in this codebase.

## Writing Style

- Do not use em dashes (—) when writing. Use alternative punctuation such as commas, parentheses, colons, or separate sentences instead.

## Code Style

- In JavaScript and TypeScript, prefer JSDoc-style block comments (`/** ... */`) over stacked single-line (`//`) comments for multi-line comments.
